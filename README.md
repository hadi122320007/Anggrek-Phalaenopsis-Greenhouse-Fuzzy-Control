# Anggrek-Phalaenopsis-Greenhouse-Fuzzy-Control
Pengembangan sistem kontrol suhu udara dan kelembapan udara menggunakan metode Fuzzy Mamdani pada mini greenhouse untuk mengatur kondisi mikroklimat pada anggrek phalaenopsis Sp. 

## Sistem yang Digunakan

- Mikrokontroler ESP32
- Sensor SHT
- Sensor soil moisture
- Kipas DC
- Growlight
- Pompa DC 

## Metode Kontrol

Metode kontrol yang digunakan adalah Fuzzy Logic Mamdani dengan input:

- Suhu udara
- Kelembapan udara

Output:

- PWM Kipas in
- PWM Kipas out

## Parameter Lingkungan

Suhu udara
- Siang hari : 27◦C – 30◦C
- Malam hari : 21◦C - 24◦C

Kelembapan udara : 60% - 80 %

Intensitas cahaya : 3.000 – 7.000 lux
